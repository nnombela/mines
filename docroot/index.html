Type your screen name and login. Create a new game session and ask others to join it. Or you may search for a session already created by other player. When ready, press the smile button and wait for the game to start. By default, you play the FLAG game mode in which you score one point for any flag you set correctly and you loose all your points for any mistake you make.

This is an early beta version. May not work as expected. Please contact me at nnombela@gmail.com if you find any problem or want to make a suggestion.
