Help & Game Rules

You can play single-player like the original game or go multiplayer by login on the server. After login, you can either create a new session or search an existing one to join. After joining a session you have a tab where you can chat with others players. There is also a player tab where you can see who is online.

Press the smile button to get ready. If you don't do it and the creator is waiting for you, the game will set you ready and start automatically. Don't go crazy uncovering cells at the beginning, trying to open the board blindly, as the server will do it for you every one second until one of the players uncover a cell. Also, after this open stage, the server will uncover a cell after five seconds if none of the players uncover or set a flag on one. In other words don't uncover or set a flag if you are not sure of it.

There are two game modes: FLAG and UNCOVER.

  • For the FLAG mode, which is the default, you score one point anytime you set a flag on a mine. If the flag is set incorrectly you loose all your points, and also you will be paused for a few seconds, afterwards you can join the game again by pressing the smile button once again

    In this mode, you can not unset a flag, as for sure It will be correctly set on a mine. The flags that you set will be red and the flags set by others will appear blue.

  • For the UNCOVER mode, which is for the ones who do'nt like to set flags, you score one point for any cell uncovered that don't have a mine. In which case you loose all the points and you will be paused for a few seconds, afterwards you can join again by pressing the smile button.

    In this mode, the flags that you set will not be send to the server and will not appear set for the rest of the players. Also as it can be incorrectly set, you can unset it anytime.

    If you uncover a cell that do'nt have any adjacent mine all the adjacent cell will be automatically uncovered, just as the original game, and you will collect one point for each one.

The Auto-flag option.

With the auto-flag option on, flags will be automatically set around uncovered cells that have as many adjacents as mines. That make the game very fast by clicking the right and left buttons simultaneously.

Developer notes

I made this game with some goals ahead, which are:

  • Very light applet in terms of low cpu usage and small memory footprint. This is important as minesweeper is usually played while doing something else. A lot of image caching and profiling has been made to achieve it
  • Designed for responsiveness. I have a pretty good record on this game, so if the responsiveness wasn't good enough the game will crap
  • Implementation of the double click technique, which is a must to break the 100 seconds barrier (I think)
  • Fun. Let more than 2 players play at the same time
  • Themeable. Ok, right now there is only one theme, but I hope to add more soon.
